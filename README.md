# React-Tic-Tac-Toe-Game
This is a simple Tic-Tac-Toe game made using ReactJS in order to have some hands-on beginner experience.
It has been deployed in gh-pages @ https://ayushmantripathy9.github.io/Tic-Tac-Toe-Game/

For local development purposes: 
### Clone this repo.
### Edit the package.json and change the homepage field to fit your gh-username
### npm start
Runs the local development server on localhost:3000 where you can view it in dev mode
### npm install gh-pages --save-dev
Installs dependencies required for deployment in github-pages.<br /> This installs the GitHub Pages package as a dev-dependency.

### npm run deploy
To deploy this app to your github-pages.<br /><br /> 
Hope you enjoyed viewing my first basic app and deployed it too !!
